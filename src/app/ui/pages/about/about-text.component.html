You must login to use most of the DMA features, as personalized storage does not work otherwise.

The Dungeon Master Assistant (DMA) is a tool for Dungeon Masters to manage D&D 5e games.

It is currently in its very early stages. It currently provides the following features:

  • Miniatures: Provide information about many D&D miniatures and allow to manage miniature collections (locations, count, search).
  • Campaigns: Add information about campaigns, adventures and individual encounters. This gives DM necessary information about monsters, items or spells while playing.
  • Map Screen: Predefined map for some locations that can be easily controller and displayed on a separate screen - for example a TV lying flat on the table as battle map. The map supports layers to allow room by room discovery of locations.
  • Campaign Screen: Show vital campaign information on a seperate screen to give players additional information about play:"
    • The rough time via a sun/moon display, including moon phases.
    • The date of the campaign, in dale reckoning.
    • The number of the round if a battle is happening.
    • A context specic image, relevant to the current play. This can be either an url manually entered or an image selected from a monster, item, or encounter.

Most of the feature require a Google account in order to be able to store any data.

DMA is currently in ALPHA stage. Expect many changes, broken things and potential data loss.
